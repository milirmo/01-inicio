¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas. HTML

El Lenguaje de Marcado de Hipertexto (HTML) es un código que se utiliza para presentar las páginas web. Se dice que este es un lenguaje de marcado debido a que contiene una serie de elementos que permiten encerrar el contenido, haciendo que se vean de cierta manera.

Etiquetas:

"head" encabezado. "meta" permite configurar la página (va dentro del head). "title" titulo (dentro de head). "body" espacio donde se programa. "p" separar párrafos. "h1" tamaño de texto 1 (el más grande). "h2" tamaño de texto 2. "h3" tamaño de texto 3. "h4" tamaño de texto 4. "h5" tamaño de texto 5. "h6" tamaño de texto 6 (el más chico).

¿Cómo es la anatomía de un elemento HTML? Dar un ejemplo con la etiqueta "p"

etiqueta apertura: se conforma de los signos "< >", y en medio contiene el nombre del elemento.

etiqueta cierre: se conforma de los signos "< >", pero en medio contiene una barra "/" seguida del nombre del elemento.

contenido: es lo que va en medio de las etiquetas de apertura y cierre.

elementos = etiqueta apertura + etiqueta de cierre + contenido

ejemplo: <p> quiero mil paquetes de papas. <p>

¿Cuál es la anatomía básica del HTML? Escribir su código y definir <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

la anatomía básica de HTML es la siguiente:


<!DOCTYPE html> (inicio de tipo de formato)
<html lang="en"> (lenguaje)
<head> (comienzo de encabezado)
<meta charset="UTF-8"> (configuración)
<meta http-equiv="X-UA-Compatible" content="IE=edge"> (configuración)
<meta name="viewport" content="width=device-width, initial-scale=1.0"> (configuración)
<title>Document</title> (título)
</head> (fin de encabezado)
<body> (comienzo de cuerpo)
</body> (fin de cuerpo)
</html> (fin del tipo de formato)

¿Qué es y que función cumple el CSS?

El CSS es un lenguaje informático que funciona para hacer que los documentos que se vayan a presentar al usuario sean atractivos y estéticos. Este lenguaje permite, por ejemplo, cambiar el color y el tamaño de los encabezados y los enlaces, insertar un diseño e incluso crear efectos de animación.

¿Cómo se incorpora el CSS dentro del HTML?

El CSS nos permite, junto con el HTML, formar una página web mejor diseñada, con más estética (tamaños de texto, colores de fondo y tipos de letra).

¿Cuál es la anatomía de una regla CSS?

la anatomía de una regla CSS es la siguiente:

h1 {

color: red;

font-size: 5em;

}

¿Cuáles son los tipos de selectores que puede utilizar CSS?

Los selectores indican cual será el elemento sobre el que se diseñará.

selectores básicos: de tipo, de clase, de ID, universal y selector de atributo.

selectores combinadores: de hermanos adyacentes, general de hermanos, de hijo, de descendientes, de columna.

selectores pseudoclases: selecciona elementos mediante información fuera de documentos.

¿Qué es javascript? ¿Podrías definirlo como un lenguaje de programación? ¿De qué tipo? Justificar.

JavaScript es un lenguaje de programación interpretativo que permite implementar funciones complejas en páginas web. Muestra distintas actualizaciones de contenido, mapas interactivos, animación de gráficos. También permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes, etc.

¿A qué se llama DOM y cual es su función?

DOM es una API del navegador que permite a los programadores comprender HTML y CSS, y que estos comprendan lo que las personas que programan quieren decir, y así les permite poder crear, eliminar y cambiar el HTML, aplicar dinámicamente nuevos estilos a la página, etc.

¿En qué puede ayudarnos JavaScript en una página web?

JavaScript puede ayudarnos para modificar dinámicamente HTML y CSS para actualizar una interfaz de usuario, utilizando la API del DOM.
